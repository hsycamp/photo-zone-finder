# photo-zone-finder

지역별 포토존을 공유하는 웹 애플리케이션

## Demo

[Link](https://photo-zone-finder.herokuapp.com)

## 주요기능

- JWT 토큰 발급 방식으로 로그인 구현
- OAuth 로그인(구글 로그인) 구현
- 게시물과 댓글 CRUD 구현
- 사진 업로드(AWS S3 Storage 를 이미지 storage로 사용)
- 좋아요 기능 구현
- 게시물 조회(좋아요순/업로드 날짜순)
